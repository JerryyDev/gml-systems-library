# 🖥️ Menu System (GML)

Functional menu system with keyboard navigation and resolution settings, ready to use in 2D games.

---

---

## 🚀 Features

- Keyboard navigation (↑ ↓)  
- Selection with ENTER  
- Settings submenu  
- Resolution switching  
- Back navigation with ESC  
- Simple and easy-to-adapt structure

## 📂 Structure

- `obj_menu` → controls the entire menu system  

### States:
- `0` → Main Menu  
- `1` → Settings  

---

## 🎮 Controls

- ↑ ↓ → Navigate  
- ENTER → Select  
- ESC → Back  

---

## 💡 How it works

The system uses a state-based structure to switch between the main menu and the settings menu.

Menu options are stored in arrays, making the system easy to expand and customize.
